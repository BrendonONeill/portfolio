FFVII website

This website was created as an experiment to test my skills in using Flexbox and CSS Grid for layout design without relying on any frameworks, while using custom CSS. The style of the site is inspired by the user interface of *Final Fantasy VII* on the PlayStation 1, aiming to replicate the nostalgic look and feel of the game's UI and the manual that accompanied PS1 games. Though it's a simple site, it successfully captures the retro aesthetic while serving as a technical exercise in building layouts from scratch using CSS
